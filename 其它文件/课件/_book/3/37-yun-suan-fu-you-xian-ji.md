## **3.7运算符优先级**

在Go语言中，一元运算符拥有最高的优先级，二元运算符的运算方向均是从左至右。

下表列出了所有运算符以及它们的优先级，由上至下代表优先级由高到低：

| **优先级** | **运算符** |
| :--- | :--- |
| **7** | **^      !   ** |
| **6** | **\*       /    %    &lt;&lt;&gt;&gt;&      &^** |
| **5** | **+      -     \|      ^** |
| **4** | **==   !=   &lt;&lt;=    &gt;=    &gt;** |
| **3** | **&lt;-** |
| **2** | **&&** |
| **1** | **\|\|** |

当然，你可以通过使用括号来临时提升某个表达式的整体运算优先级。
